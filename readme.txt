项目名称：宠物商城
项目时间：30天左右
项目描述：
    技术栈：html，css，bootstrap，jQuery，node.js，ajax，axios
    1.html，css，bootst框架完成静态页面的渲染，jquery框架及一些原生的DOM操作完成前后端的交互
    2.使用ajax向后端发送请求获取数据，
    2.使用sql完成后端数据库的搭建
    2.完成的页面：首页，登录页，注册页，商品列表页，购物车页，地址页，订单页
    3.实现的功能：显示商品，添加到购物车，购物车商品结算，分页查询，关键词搜索，楼层效果，放大镜等功能
项目遇到的问题：
    1.在将商品添加到购物车时，不知道该商品是否存在于购物车，需先查询，再添加，直接使用sql语句时，会嵌套，形成回调地狱
        解决：使用ES6新特性中的promise，控制执行顺序，在添加商品之前先查询购物车列表
    2.当商品添加到购物车时，不知道该往哪位用户的购物车添加
        解决：使用session会话功能，当用户登录时，用session记录登录用户的id，当添加商品时，可同时添加一条用户id的信息
项目收获：
    1.熟悉jQuery及一些原生DOM的使用
    2.进一步熟悉ES6中promise的使用
    3.对session的工作原理也有进一步的了解
